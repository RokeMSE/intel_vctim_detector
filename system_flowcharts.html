Inspection Workflows

Industrial Quality Control & Defect Detection System

Workflow 1: VCTIM Inspection

Human User
Image Acquisition
User captures all VCTIMs (entire Test Socket PCB) with camera and inputs into TRB PC to access the local system
TRB PC
Inference
System infers the total amount of VCTIMs and identifies missing VCTIMs in the captures
Traceability (if needed): Decodes/Manual input/OCR the serial number to log the unit into the database
Output to Floor
Defect Map Display
A "Defect Map" is displayed on the floor monitor, highlighting whether a unit is missing VCTIM or not along side a confident score for the operator to inspect

Workflow 2: Pin Defect Detection

Human User
Image Acquisition
User uses high-resolution station camera to capture each Socket and input into TRB PC to access the local system
TRB PC
Patching & Embedding
Crops 1,000+ tiny patches and generates ResNet18 embeddings for each
TRB PC
Anomaly Detection
Multi-stage defect detection process
The "No-Miss" Check: Every coordinate is scored
Missing/Bent/Burnt/etc. Logic: If a coordinate score exceeds the threshold, it is flagged
Potential undocumented defects found
Output to Floor
Defect Map Display
A "Defect Map" is displayed on the floor monitor, highlighting specific pin coordinates for the operator to inspect
